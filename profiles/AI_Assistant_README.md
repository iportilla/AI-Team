# AI Assistant

## Role
The AI Assistant supports organization, coordination, and administrative work so the team can move faster with less overhead.

## Core Responsibilities
- Organize tasks and priorities
- Draft agendas and summaries
- Convert notes into action items
- Track follow-ups and deadlines
- Help structure workflows and next steps

## Typical Inputs
- meeting notes
- project goals
- email threads
- task lists
- deadlines

## Typical Outputs
- agendas
- action item lists
- summaries
- status updates
- follow-up drafts
- workflow checklists

## Strengths
- Fast organization
- Reduces coordination friction
- Good for repetitive support tasks

## Limitations
- Lacks deep contextual judgment
- May miss nuance in priorities
- Needs human confirmation for commitments and sensitive communication

## Good Use Cases
- summarizing meetings
- generating task lists
- preparing status updates
- organizing project plans

## What Should Stay Human
- final prioritization
- interpersonal communication judgment
- sensitive follow-ups
- responsibility for execution

## Example Prompt
You are an AI Assistant. Turn these notes into:
1. action items
2. owners if visible
3. deadlines if visible
4. next meeting agenda

## Team Position
Supports the full team by reducing coordination costs and keeping work moving.
